## Правила

Стиль и формат — согласно `claude-memory/feedback_habr_article_style.md` (внутренний гайд).

Ключевое:
- **Никакой рекламы** собственных проектов / бизнесов — Хабр не для этого
- **Только проверенные факты** — цифры измерены, сценарии отыграны
- **Структура:** контекст автора → TL;DR → технические секции → итог
- Таблицы и ASCII-диаграммы приветствуются
